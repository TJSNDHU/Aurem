# Constitution pointer

The canonical constitution lives at:

→ **`/app/.specify/memory/constitution.md`**

It contains the founder's hard rules (Hinglish only, production-always,
no mocks, route-level fixes, regression tests mandatory, etc.). Every
agent — main, fork, Claude Code, Copilot, Cursor — must load it before
acting.

This file exists so the constitution is discoverable from both the
`/app/memory/` (AUREM internal) and `/app/.specify/memory/` (spec-kit
compatible) paths.
