# Thesys Agent Builder — Reference for AUREM
**Saved for future use**

- **URL**: https://www.thesys.dev/agent-builder
- **What**: No-code AI agent builder with **Generative UI** (agents respond with charts, forms, cards — not just text)
- **Going open source** on GitHub
- **Powered by C1** (their UI generation engine)

## Key Differentiator
Normal chatbots return **text**. Thesys agents return **interactive UI** — charts, product cards, comparison tables, forms that users can act on immediately. Conversations → real actions.

## How It Works (3 Steps)
1. Upload data / Connect MCPs (files, URLs, databases, tools)
2. Customize agent (tone, instructions, conversation starters, brand styling)
3. Share or embed (live link or embed snippet)

## AUREM Relevance
- **ORA Chat upgrade path**: Instead of text-only responses, ORA could generate interactive UI (charts of pipeline data, deal cards, revenue graphs) directly in the chat
- **Generative UI concept**: Thesys's C1 engine is going open-source — could be integrated into AUREM's frontend for rich agent responses
- **Agent Builder for customers**: AUREM could offer a "Build Your Own Agent" feature where customers create custom agents for their specific workflows
- **Template marketplace**: Thesys's community template model could inspire an AUREM Agent Template Store
- **Embed model**: Their embed-anywhere approach is exactly what AUREM's Shopify ORA Chat widget does — validate that direction
