# ORA Learning Meta
## Purpose
ORA continuously improves its skills based on real outreach outcomes.
Every day at 2AM UTC, ORA analyzes what worked, what didn't, and updates
its own skill files with evidence-based insights.
## What ORA learns
- Which channels work best per business category
- Which tone converts best per lead profile
- Which drip step generates the most responses
- Which cities/regions respond fastest
## Rules for learning
- Minimum 10 data points before any insight
- Never speculate — only data-backed updates
- Never remove existing skill content
- Always timestamp learnings
- Learning insights visible in Morning Brief
